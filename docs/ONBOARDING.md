## Step 8: Domain & SSL Setup üåê

### Option A: Use MOSAIC Subdomain (Free)

**Automatic Setup**:
- Your site is instantly available at: `yoursite.mosaic.app`
- Free SSL certificate included
- No DNS configuration needed
- Perfect for testing and personal sites

### Option B: Connect Custom Domain (Pro+)

#### Step-by-Step Domain Connection

1. **Add Domain**:
   - Go to Settings ‚Üí Domains
   - Click "Add Custom Domain"
   - Enter: `www.yourdomain.com`
   - Click "Verify"

2. **DNS Configuration**:
   
   **If using www subdomain**:
   ```
   Type: CNAME
   Name: www
   Value: sites.mosaic.app
   TTL: 3600 (or default)
   ```

   **If using root domain**:
   ```
   Type: A
   Name: @
   Value: [IP address will be provided]
   TTL: 3600
   
   Type: A
   Name: @
   Value: [Secondary IP will be provided]
   TTL: 3600
   ```
   
   **Note**: Actual IP addresses will be provided in your MOSAIC dashboard under Settings ‚Üí Domains.

3. **Popular DNS Providers**:

   **GoDaddy**:
   - Login to GoDaddy
   - My Products ‚Üí Domains
   - DNS ‚Üí Manage Zones
   - Add Record

   **Namecheap**:
   - Domain List ‚Üí Manage
   - Advanced DNS
   - Add New Record

   **Cloudflare**:
   - Select domain
   - DNS ‚Üí Add Record
   - Ensure proxy OFF (cloud grey)

4. **Verification**:
   - DNS changes take 5 minutes - 48 hours
   - MOSAIC checks automatically every 5 minutes
   - You'll receive email when verified
   - Green checkmark appears in dashboard

5. **SSL Certificate**:
   - Automatic Let's Encrypt certificate
   - Issued after DNS verification
   - Auto-renewal every 90 days
   - HTTPS enforced automatically

### Multiple Domains (Business+)

Map multiple domains to same site:
- `www.yourdomain.com` (primary)
- `yourdomain.com` (redirect to www)
- `old-domain.com` (redirect)

## Troubleshooting üîß

### Domain Not Connecting

**Issue**: Custom domain shows "Not Verified"

**Solutions**:
1. Wait 24-48 hours for DNS propagation
2. Check DNS records with [DNS Checker](https://dnschecker.org)
3. Verify CNAME/A records exactly match instructions
4. Remove conflicting records (old A or CNAME for same name)
5. If using Cloudflare, disable proxy (grey cloud)
6. Contact support with domain name

### SSL Certificate Pending

**Issue**: Site shows "Not Secure" warning

**Solutions**:
1. Wait 15-30 minutes after DNS verification
2. Clear browser cache and retry
3. Check domain is verified first
4. Visit HTTPS version explicitly: `https://yourdomain.com`
5. Contact support if pending >24 hours

### Slow Page Loading

**Issue**: Site loads slowly

**Solutions**:
1. Compress images (use TinyPNG or similar)
2. Limit custom CSS/JavaScript
3. Reduce number of fonts loaded
4. Check analytics for bottlenecks
5. Upgrade plan if bandwidth exceeded
6. Contact support for performance review

### Content Not Saving

**Issue**: Changes don't persist after saving

**Solutions**:
1. Check internet connection
2. Disable browser extensions (try incognito)
3. Clear browser cache
4. Try different browser
5. Check if another user is editing simultaneously
6. Contact support with browser console errors
